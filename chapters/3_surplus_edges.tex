We will prove weak convergence by showing that
\begin{equation}
\Exp{ f(\bar{Z}^t_n, \bar{N}_n) } \longrightarrow \Exp{ f(W^t, N) }
\end{equation}
as $n \rightarrow \infty$, 
for all continuous, bounded functions 
$f: D^2 \rightarrow \Real$.
